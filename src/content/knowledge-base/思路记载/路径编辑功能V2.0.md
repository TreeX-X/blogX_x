# 视图机制

编辑模式下锁定到 3D模式 下进行编辑

# 截断

V1.0 流程:
1. [UI]   点击截断功能——唤出编辑向导

2. [UI]   调用VTK状态处理函数 (true)， [VTK]进入截断模式

3. [VTK]截断模式下
   对"编辑图层"可进行 选中图元，图元"特殊"高亮切换

   [UI]"ESC可以直接退出"

4. [VTK]基于现有锚点机制开
   选中锚点——锚点特殊常态显示：
   返回"图元信息，锚点信息"
    [UI]"ESC可以直接退出"

5. [UI]   根据锚点，根据后端机制，重新构造数据 ，自动调用VTK截断状态函数（false）

6. [VTK&UI]退出截断模式
