# 前置技术

开发AI工作流：
VibeCoding + SpecCoding + Harness Design

技术栈：
astro框架 + lanceDB 数据库

# 开发流程记录

## 构建框架

```
npm create astro@latest my-vibe-blog

cd my-vibe-blog

npx astro add tailwind

npm install @tailwindcss/typography

npm run dev
```

## 开启后端API能力

安装Vercel 官方适配器（Adapter）

```
npx astro add vercel
```

LANCEDB_URI="db://你的链接"

LANCEDB_API_KEY="sk_你的LanceDB秘钥"

SF_TOKEN="hf_你的SiliconFlow秘钥"

```
# 安装 LanceDB 官方包 和 HuggingFace 官方接口包

npm install @lancedb/lancedb @huggingface/inference

# 安装本地解析 Markdown 需要的工具包（稍后写脚本会用到）

npm install dotenv gray-matter
```
