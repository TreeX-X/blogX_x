# 依赖工具链：
vscode - copilot + Agent + Workflow

# 思路记录

在Agent每次对话结束时，调用copilot-hook调用.sh脚本执行自动化任务

.sh 脚本功能细节:
1. 记录git 工作树的具体代码变更 和 本次对话提出的需求和功能
2. 对记录的内容,放入window缓存目录中,命名为"log.md"
3. log.md 具备以下记录:
~~~
   - 记录当天的日期
   - 对话产生的git工作树代码变化
   - 对话提出的开发需求和功能
~~~
4. 对.md的日期进行检查,当日期对不上时,则证明是下一天的访问, 自动清空.md的内容
5. 当日期检查通过时,则把每次记录的内容,按顺序放入.md中
